Privacy Policy

Last updated: April 18, 2026

1. Data Collection

The Chip Slicer Pro visual for Power BI does not collect, store, or transmit any personal information, user data, or business report data outside of your Power BI environment.

2. Data Processing

All data processing is performed locally within the user's Power BI infrastructure. The visual only uses data provided by the Power BI platform to render the filter elements (chips/pills).

3. Third-Party Services

This visual does not make calls to external servers and does not use any third-party services. It strictly complies with Microsoft's security policies for certified visuals.

4. Licensing

License management is handled exclusively through Microsoft's official Power BI licensing API. No external authentication or payment processing is performed by this visual.
