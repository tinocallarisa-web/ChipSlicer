Política de Privacidad

Última actualización: 18 de abril de 2026

1. Recopilación de Información

El visual Chip Slicer Pro para Power BI no recopila, almacena ni transmite ningún tipo de información personal, datos del usuario o datos del informe empresarial fuera de su entorno de Power BI.

2. Procesamiento de Datos

Todo el procesamiento de datos se realiza localmente dentro de la infraestructura de Power BI del usuario. El visual solo utiliza los datos proporcionados por la plataforma Power BI para renderizar los elementos visuales de filtrado (chips/pills).

3. Servicios de Terceros

Este visual no realiza llamadas a servidores externos ni utiliza servicios de terceros. Cumple estrictamente con las políticas de seguridad de Microsoft para visuales certificados.

4. Contacto

Si tiene alguna duda sobre esta política de privacidad, puede contactarnos en: support@tcviz.com
